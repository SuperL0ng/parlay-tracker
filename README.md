# Parlay Tracker

This repository contains one canonical application source for both domains.

- Source: `app/src`
- Build identities: `app/config/builds.json`
- Generator: `scripts/build-static.mjs`
- Gold output: `build/gold` for `simonsports.bet`
- Silver output: `build/silver` for `simonsportsbetting.com`

Generated deployment output is not committed to the development branch. See [the hosting workflow](docs/HOSTING_WORKFLOW.md).
